Random Password vs Passphrase

Both can be strong. The better choice depends on whether you need to memorize, type, share, or store the secret.

Simple rule

Use random passwords for accounts saved in a password manager. Use passphrases when a human needs to remember or type the secret more often, such as a master password or some Wi-Fi setups.
