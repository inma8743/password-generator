What is a passphrase?

A passphrase is a password made from several random words. It can be easier to remember than a random string while still being strong when it is long enough.

When is it useful?

  • Master passwords you need to type by hand.
  • Wi-Fi, device, and local encryption passwords that should be memorable.
  • Accounts where a long password is allowed and readability matters.
