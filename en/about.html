About Password Generator

Password Generator is a free browser-based security tool for creating strong random passwords and passphrases. Generated password strings are not sent to or stored on the operator's server.

Purpose

This site helps visitors create random passwords, passphrases, and account security checklists without installing an app. It also connects related security, developer, writing, calculator, and market-research tools.

Security Principles

  • Password generation uses crypto.getRandomValues in the browser.
  • Generated password strings and selected options are not stored on the operator's server.
  • Analytics events do not include generated password strings.
  • For important accounts, use at least 16 characters and a unique password per account.
  • Store generated passwords in a trusted password manager.
