Password safety tips

  • Use at least 16 characters for important accounts.
  • Use a different password for every website.
  • Use a password manager and two-factor authentication when possible.
  • Avoid personal information such as names, birthdays, and phone numbers.

How it works

This tool uses crypto.getRandomValues in your browser. The password is generated locally and only appears on your screen.

When should you create a new password?

Create a new password when you reused the same password on multiple websites, received a breach alert, or still rely on an old memorable password. For email, banking, cloud storage, and work accounts, a longer random password stored in a password manager is usually safer than a short password you can remember.

How to store it safely

After copying the password, save it in a trusted password manager instead of a note, chat message, or screenshot. Turn on two-factor authentication when the service supports it, especially for accounts that can reset other passwords.

Frequently asked questions

Are generated passwords sent to a server?
No. Passwords are generated locally in your browser and are not sent to or stored on a server.

How long should a strong password be?
Use at least 16 characters for important accounts, and use a different password for every website.
